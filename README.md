# Laravel Vue Inertia SPA laravel-inertia-vue-intro-01

A simple **Single Page Application (SPA)** built with Laravel, Vue 3, and Inertia.js.  
Uses **Laravel Sail** with **PostgreSQL** in a local Docker container for development.

---

## Features

- Laravel 12 backend
- Vue 3 frontend with Composition API
- Inertia.js SPA routing
- PostgreSQL database
- Tailwind CSS styling
- Dockerized local development with Sail

---

## Requirements

- Docker & Docker Compose
- PHP >= 8.2
- Composer
- Node.js >= 18
